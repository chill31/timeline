Chill31

Anybody is free to use this timeline. This template uses marvel movies as a reference to create the timeline. Change the .container element's class from vertical to horizontal to make the timeline go from left to right instead of top to bottom. There are a lot more features than direction only, so go to the not so descriptive documentation to find out more. There's also the Github README


The events may not be in order, as this is for the movies release date. For example, the Eternals movie is set in 5000 BC, but it was released in 2021, so in this timeline, the movie will be in the 2021 Card.
